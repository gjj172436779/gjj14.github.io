乐队简介

ONE OK ROCK（ワンオクロック）是一支日本摇滚乐队，于2005年成立，2007年正式出道。凭借扎实的音乐功底，前沿的曲风和热血激昂的现场表现力已稳固发展成为地位超群的实力团体。乐队现成员包括主唱Taka、吉他手Toru、贝斯手Ryota、鼓手Tomoya四人

part 1

2005年4月，堀越高等学校的山下亨（Toru）、小浜良太（Ryota）、鬼泽亚历山大礼门（Alex）、小柳友（You）四人组成了摇滚乐队ONE OK ROCK。5月，Toru邀请森内贵宽（Taka）加入，担任主唱。7月，在NEW POWER GENERATION Vol.3进行了第一次现场表演，之后活跃于东京地区进行了50多场LIVE演出2006年6月30日，小柳友为专心演艺之路而离开乐队，鼓手的职位暂由神吉智也（Tomoya）担任。7月26日，发行自由迷你专辑「ONE OK ROCK」，作为一支实力型音乐团体倍受关注。8月，参加夏日祭，在关东近郊举办LIVE演出.2007年，加盟Amuse事务所，Tomoya正式加入，正式出道。12月开始第一次巡演「世の中シュレッダー」（世界碎纸机）。2009年4月5日，吉他手Alex因在轨道列车中涉嫌猥亵女性而被警方逮捕，导致乐队原本预定为电视剧《天生妙手》演唱的主题曲遭到更换[2] ，正在进行的全国巡演「ONE OK ROCK 2009 “Emotion Effect” TOUR」中止，5月13日，Alex宣布脱退，ONE OK ROCK成为四人乐队，由Toru担任第一吉他手，将已有的歌曲重新编曲为单吉他的形式。9月，ONE OK ROCK开启新的巡演「ONE OK ROCK 2009 “Overcome Emotion” TOUR」，以四人姿态重新登场。
